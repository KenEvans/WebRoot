  1. Android Sensor Overview
    Sensor documentation created and maintained by Google for Android developers. Excellent information on what sensors are available and how to get to them. Includes code examples, videos, and a best practices section.
    Rating: 5 out of 5

  2. Galaxy S5 Sensors
    A description and brief overview of the sensors available on the Samsung Galaxy S5. Includes a graphic showing the sensor location on the motherboard/case.
    Rating: 4 out of 5

  3. Tutlane Android Sensor Tutorial
    This site includes an overview followed by code examples in Java and XML. Generally, this information can be found on the "Android Sensor Overview" site.
    Rating: 4 out of 5

  4. Tutorials Point Android Sensor Tutorial
    This site includes code examples in Java and XML. The tutorial assumes that you have been building an application started in previous sections. There is a 7 1/2 hour video tutorial class for Android programming available for $10. Generally, this information can be found on the "Android Sensor Overview" site.
    Rating: 3 out of 5

  5. Mozilla Developer Net Web Docs
    Tools, tutorials and information about coding for Mozilla browsers. Much of the information is also cross platform.
    Rating: 5 out of 5

  6. Apple Core Motion
    Documentation for the Apple Core Motion Framework for all the motion related sensors. Very dry reading; I didn't like this site. Other than declarations and parameter explanations, I did not find many code examples.
    Rating: 2 out of 5
